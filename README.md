### how to play?
Unscramble the letters correctly. Complete ten questions to win!

### how to install?
Download `dist/pomenohenn.exe`

**Building for other environments**
You need to run `pip install` on `pyxel` and `pyinstaller`

`pyxelpackager main.py`
